- User pink for buttons color for better accessibility.

- Tooltips added to icons for clarity.

**Quiz Card Shadows:**

- Hover effects highlight selected quiz, improving usability.

**Navigation Bar:**

- Icons and text labels make navigation intuitive and accessible.

**Alt Text for Images:**

- Descriptive alt text improves screen reader support and accessibility.

- Asterisks indicate mandatory inputs, enhancing clarity.

**Error Modals:**

- Clear, persistent error messages help users understand and correct input mistakes.

**Focused Inputs:**

- Labels animate and change color on focus, improving accessibility.

**Keyboard Support:**

- Enter key submits forms across login, registration, and join game pages.
